# Proyecto Final Chiquitibum

## Descripción

Este repositorio contiene el proyecto final del equipo Chiquitibum para la materia de Análisis y Diseño de Sistemas de Información.

El proyecto consiste en diseñar una aplicación para programar cafés antes de entrar a clase.

La aplicación debe permitir que el usuario pueda:

* Seleccionar el menú.
* Elegir un pedido.
* Escoger una opción de pago.
* Confirmar la hora de entrega.

## Funcionalidades requeridas

* Menú de productos.
* Selección de pedido.
* Tres opciones de pago.
* Confirmación de hora de entrega.

## Entregables
